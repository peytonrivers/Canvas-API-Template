# Canvas-API-Template
1. Download the file
2. Input your Canvas Token
3. Run on Local Host
to run on local host follow these steps, copy and past one by one into integrated terminal
1. python3 -m venv venv
2. source venv/bin/activate
3. pip install fastapi "uvicorn[standard]"
4. Pip install requests
5. uvicorn template:app --reload
